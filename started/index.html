Getting Started

To get started, you will need a compiler and text editor. There currently exist 4 nice text editors available with syntax highlighting. ooc-mode for Emacs is the most advanced one, featuring on-the-fly correction using flymake, automatic tab indentation, templates for unfolding common constructs, and a completion system in the works.

download   source rock ooc compiler

To install rock, it is recommended to fetch the bootstrap from alpaca and run make.

{% highlight bash %} $ wget http://commondatastorage.googleapis.com/rock-linux/rock-bootstrap.tar.bz2 $ tar xjf rock-bootstrap.tar.bz2 $ cd rock $ make {% endhighlight %}

Don't forget! Add bin/ to your $PATH after compilation
